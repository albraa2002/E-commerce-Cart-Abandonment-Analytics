🛒 E-commerce Funnel & Cart Abandonment Analytics

📌 Project Overview
This project focuses on optimizing the e-commerce customer journey and identifying revenue leaks in the sales funnel. By analyzing a simulated dataset of 20,000 user sessions across an entire year, this dashboard provides actionable insights into conversion rates, cart abandonment behaviors, and traffic source quality.

🚀 Key Business Insights
Funnel Drop-off Identification: Visualized the full customer journey, revealing a steep drop-off between product views and cart additions, resulting in an overall Conversion Rate of ~5%.
Cart Abandonment Crisis: The analysis shows a massive ~50.2% overall Cart Abandonment Rate, representing significant lost revenue opportunities.
Traffic Source Quality: Data proves that while 'Paid Social' drives high traffic volume, it suffers from the highest cart abandonment rate (~70%). Conversely, 'Organic Search' yields much higher quality leads.
Revenue Tracking: Tracked the Monthly Revenue Trend alongside an Average Order Value (AOV) of EGP 1,913 to help optimize Customer Acquisition Cost (CAC).

🛠️ Tech Stack
Language: Python 3.x
Data Manipulation: Pandas, NumPy
Data Visualization: Plotly (Funnel Chart, Bar Chart, Line Chart)
UI/UX: Standalone HTML/CSS Grid Dashboard (Modern E-commerce UI)
